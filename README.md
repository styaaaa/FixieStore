# FixieStore

Modern fullstack e-commerce web application with responsive UI, product management, authentication, and shopping cart features.

## Features

- User authentication
- Product listing
- Shopping cart system
- Product CRUD
- Responsive design
- Search and filtering
- Admin dashboard

## Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MySQL

## Installation

```bash
git clone https://github.com/USERNAME/fixie-store.git
```

Frontend:
```bash
cd client
npm install
npm run dev
```

Backend:
```bash
cd server
npm install
npm start
```

## Environment Variables

Create `.env` file:

```env
DB_HOST=
DB_USER=
DB_PASSWORD=
JWT_SECRET=
```
